# ARTHA AI

Avatar-led digital wealth advisor embedded in a bank mobile app.

## Quick start

```bash
# Backend
cd artha_backend && pip install -r requirements.txt
uvicorn main:app --reload --port 8000

# Frontend
cd artha_frontend && npx serve -s . -l 3000
```

Demo auth: `Authorization: Bearer demo-token`
